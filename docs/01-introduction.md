# Sparta XR - Introduction

Sparta XR is a web platform for uploading, optimizing, managing, and sharing 3D models. Users can generate QR codes for their models and view them in augmented reality (AR) on any device.

## Vision

Make 3D model sharing as simple as sharing a link. Sparta XR enables creators, designers, and engineers to publish interactive 3D experiences without writing a single line of code.

## Key Features

- 3D Model Upload & Optimization
- Interactive WebGL Viewer
- QR Code Generation
- AR Viewing
- Analytics & Tracking
- Team Collaboration
- API Access
