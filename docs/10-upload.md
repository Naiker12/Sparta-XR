# Upload Module

## Supported Formats

- GLB (.glb)
- GLTF (.gltf)
- OBJ (.obj)
- FBX (.fbx)
- STL (.stl)
- PLY (.ply)

## Upload Flow

1. Select or drop file
2. File validation (type, size)
3. Upload with progress tracking
4. Server-side optimization
5. Thumbnail generation
6. Ready for viewer and QR generation

## Size Limits

- Free tier: 50MB
- Pro tier: 200MB
- Enterprise: 500MB
